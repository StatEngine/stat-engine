Ownerhsip

Use the dropdown to modify any user ownerships.
  • GRANT OWNERSHIP will allow make the user a co-owner. This allows the user ability to grant/revoke permissions, as well as delete a workspace.
  • REVOKE OWNERSHIP will remove the user as co-owner.

Access Permissions

Use the dropdown to modify a user's access.

  • GRANT READ will allow ther user read access over a workspace's saved objects. This allows the user to filter, and explore dashboards and visualizations. However, the user will NOT be able to create or modify any dasbhoards or visualizations in this workspace.
  • GRANT WRITE will allow ther user write access over a workspace's saved objects. This allows the user to create, edit, and delete dashboards and visualizations. The user will NOT have some workspace adminstritator capabilities, such as changing advanced settings or exporting objects and reports.
  • GRANT FULL will allow the user full access over a workspace's saved objects. This allows the user to create, edit, and delete dashboards and visualizations. The user will also have workspace administrator capabilities, such as changing advanced settings or exporting objects and reports.
  • REVOKE ACCESS will remove the user's access from the workspace. No saved objects will be deleted.
